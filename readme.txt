uyghur ULY Permalinks (ئۇيغۇرچە مۇقىم ئۇلانمىلا)

== Description ==
1.بۇ قىستۇرما يازما تىمىسى، سەھىپە، خەتكۈچ قاتارلىقلارنىڭ «مۇقىم ئۇلانمىسى»،ياكى «باشقا نام»نى توشقىزمىسىڭىزمۇ ئاپتۇماتىك ھالدا ULY غا ئايلاندۇرۇپ بىرىدۇ .
2. ھەمدە بۇ بۇندىن كېيىن مەشغۇلاتلارغا كۇچكە ئىگە .
3.بۇندىن بۇرۇنقىنى ئۆزگەرتىش ئۈچۈن توپ ئۆزگەرتىش ئىقتىدارى قىتىلدى.بىراق بۇندىن بۇرۇنقىلىرىنى ئۆزگەرتىشنى تەشەببۇس قىلمايمىز .
4.بۇ قىستۇرمىنڭ نورمال ئىشلىشى ئۇچۇن سىز چوقۇم <a href="/wp-admin/options-permalink.php">مۇقىم ئۇلانما(permalink)</a> نىڭ ئىچىدىن بەزى تەڭشەشلەرنى ئىلىپ بىرىشىڭىز كىرەك . مەسىلەن /%postname%/ ،/%postname%.html دىگەندەك .يەنى سىز چوقۇم %postname% نى ئىشلىتىشىڭىز كىرەك . بولمىسا بۇ قىستۇرما كۇچكە ئىگە بولمايدۇ .

== Installation ==

قىستۇرما بىتىدىن 'uyghur ULY Permalinks' نى ئىزدەش ئارقىلىق قاچىلاڭ.

ياكى

چۇشۇرۇپ '/wp-content/plugins/' نىڭ ئاستىغا كۆچۇرۇڭ
